## pure data

<div style={{ fontSize: '1.2rem', color: 'black', marginTop: '20px' }}>
  <p>
    <strong>pure data</strong> means no configuration, just dumping an array of objects into{' '}
    <i>data</i>.
  </p>
  <p>It will use the keys (of the first object) as Column Titles</p>
</div>

```javascript

  return <DataTable data={people} />
}
```

## simple columns config

<div style={{ fontSize: '1.2rem', color: 'black', marginTop: '20px' }}>
  <p>
    using a <i>columns</i> array, you can filter what columns to show
  </p>
</div>

```javascript
import DataTable from 'Components/DataTable/'

const MyComponent = ({ people }) => {
  return <DataTable data={people} columns={['name', 'email', 'city', 'creditcard']} />
}
```
